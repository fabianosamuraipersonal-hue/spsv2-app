Ajuda rápida

Fluxo V2: Config → Login → Alunos → Sessões → Itens (exercícios) → Relatórios.

Se aparecer “Could not find the 'X' column …”: seu banco não tem essa coluna. Este app tenta se adaptar automaticamente, mas se você estiver criando colunas manualmente, mantenha os nomes assim: goal, status, session_date, group_name.

Treinos regressivos (SPSV3): nos Itens da sessão você tem um campo Método e um campo Esquema. Se o banco ainda não tiver essas colunas, o sistema salva no notes automaticamente.

Prescrição SPSV3: em Alunos existe o botão Prescrever para gerar a Sessão 1 (40% CF) com FITT-VP (kcal/MET).
